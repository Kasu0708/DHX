## Document exchange protocol DHX
![](DHX.PNG)  ![](X-ROAD.PNG)

__DHX is a new document exchange protocol, that  allows Estonian publicsector agencies exchange documents by distributed principle. DHX builds on innovative features of X-Road version 6.__
